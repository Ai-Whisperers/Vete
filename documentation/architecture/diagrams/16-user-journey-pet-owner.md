## Key Touchpoints

1. **Discovery**: Find clinic, learn about services
2. **Registration**: Create account, verify email
3. **Pet Registration**: Add pet profiles
4. **Booking**: Schedule appointments
5. **Appointment**: Attend consultation
6. **Post-Visit**: Access records, pay invoices
7. **Ongoing**: Maintain pet health records
