## Instrucciones de Entrega

### Formato de Archivos
- **Fotos:** JPG o PNG
- **Logo:** PNG con fondo transparente (preferido) o SVG
- **Tamaño mínimo:** 1920x1080 px para banners, 800x600 px para otras

### Cómo Enviar
1. Cree una carpeta en Google Drive o Dropbox
2. Organice las imágenes según la estructura abajo
3. Comparta el link a: imagenes@vetepy.com

### Alternativa
Si no tiene todas las imágenes, podemos:
- Usar imágenes de stock gratuitas
- Programar una sesión de fotos (costo adicional)
- Usar las imágenes de sus redes sociales (con permiso)

## Consejos para Mejores Fotos

1. **Iluminación:** Natural es mejor, evite flash directo
2. **Limpieza:** Asegúrese que el espacio esté ordenado
3. **Mascotas:** Fotos con animales siempre son más atractivas
4. **Calidad:** Use cámara de celular moderno como mínimo
5. **Orientación:** Horizontal para banners, cuadrado para perfiles
6. **Consistencia:** Mantenga un estilo similar en todas las fotos
