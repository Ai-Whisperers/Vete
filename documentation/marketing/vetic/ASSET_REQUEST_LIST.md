## Proceso de Solicitud

### Para Screenshots de Producto
1. Definir qué vista/módulo se necesita
2. Preparar datos de demo en staging
3. Capturar en resoluciones especificadas
4. Anotar si se necesita edición (blur datos, highlight áreas)
5. Entregar en formato original + editado

### Para Diseño Gráfico
1. Completar brief: objetivo, audiencia, mensaje clave
2. Proveer copy final (no placeholder)
3. Indicar dimensiones y formato de entrega
4. Revisar primera versión en 48h
5. Máximo 2 rondas de revisión

### Para Video
1. Definir objetivo y duración target
2. Escribir guión/storyboard
3. Grabar material crudo
4. Primera edición para revisión
5. Incorporar feedback
6. Render final con subtítulos
