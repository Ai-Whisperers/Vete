## Nano Banana Pro Best Practices

### Why Nano Banana Pro?

Nano Banana Pro excels at:

- **Text rendering** - Perfect for signage, certificates, labels
- **Character identity preservation** - Consistent team member faces across images
- **Multi-image fusion** - Combine reference photos with new contexts
- **Technical precision** - Understands camera specs, lighting setups, medical equipment

### Prompt Engineering Rules

1. **Be technically specific** - Replace vague adjectives with precise specs
   - ❌ "good lighting"
   - ✅ "three-point lighting with key light at 45°, fill at 30%, and rim light"

2. **Specify camera details** - Lens, aperture, focal length
   - ❌ "close-up photo"
   - ✅ "Canon EOS R5, 85mm f/1.8 lens, shallow depth of field, bokeh background"

3. **Include context and purpose** - Tell the model where this will be used
   - Add: "for veterinary clinic website hero banner"
   - Add: "professional headshot for medical team page"

4. **Use negative constraints** - Explicitly exclude unwanted elements
   - Add: "no text overlays, no watermarks, no graphic medical content"

5. **Reference real-world knowledge** - Nano Banana Pro has vast world knowledge
   - ✅ "modern veterinary clinic similar to VCA Animal Hospitals"
   - ✅ "IDEXX diagnostic equipment"

## Batch Generation Workflow

### In Antigravity IDE:

1. **Set up project folder** for generated images
2. **Generate hero images first** - highest visibility
3. **Generate team photos** - use reference images if available for consistency
4. **Generate facilities** - can batch similar types together
5. **Generate services** - group by category (medical, grooming, etc.)

### Quality Check After Each Generation:

- [ ] No anatomical errors on animals
- [ ] Correct breed/species as specified
- [ ] Brand colors visible where specified
- [ ] Proper aspect ratio
- [ ] No unwanted text generated
- [ ] Mood matches specification
- [ ] Resolution sufficient for web use
