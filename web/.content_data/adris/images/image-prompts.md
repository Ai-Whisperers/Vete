## Prompt Guidelines

### Style Consistency Keywords

Use these keywords across all prompts to maintain visual consistency:

**Color Palette**:

- Primary: "deep forest green (#2F5233)"
- Accent: "warm gold (#F0C808)"
- Background: "soft off-white, clean white"

**Photography Style**:

- "professional veterinary photography"
- "soft natural lighting"
- "shallow depth of field"
- "Canon EOS R5, 85mm lens"
- "warm color grading"

**Mood/Tone**:

- "warm and inviting"
- "professional and trustworthy"
- "modern veterinary clinic"
- "compassionate care"

### Technical Settings

- **Aspect Ratios**: 16:9 (heroes), 1:1 (team), 4:3 (facilities), 3:4 (service cards)
- **Quality**: Always add "high quality, 4K, detailed"
- **Style**: Add "photorealistic" unless illustration is needed

## Negative Prompts Reference

Use these negative prompts to avoid unwanted elements:

### General Negative Prompts

```
ugly, deformed, noisy, blurry, low contrast, unrealistic, oversaturated, underexposed, overexposed, grainy, distorted, disfigured, bad anatomy, wrong proportions, watermark, signature, text, logo, banner, extra limbs, missing limbs, floating limbs, disconnected limbs, mutation, mutated, disgusting, amateur, poorly lit, harsh shadows
```

### For Pet Photos

```
aggressive expression, scared animal, stressed animal, injured, wounds, blood, graphic content, sad, crying, abused, malnourished, dirty, unhealthy, matted fur, anxious, fearful eyes
```

### For Medical/Clinical Settings

```
dirty, unclean, messy, disorganized, rusty equipment, old outdated equipment, scary, intimidating, dark, gloomy, graphic surgery, blood, open wounds, distressed patient
```

### For Professional Portraits

```
unflattering angle, harsh lighting, unfriendly expression, intimidating pose, unprofessional attire, messy hair, tired expression, fake smile, awkward pose, bad cropping
```

## Prompt Modifiers by Platform

### Midjourney Specific

Add to end of prompts:

- `--ar 16:9` (or appropriate ratio)
- `--v 6` (latest version)
- `--q 2` (quality)
- `--s 250` (stylize for more artistic)
- `--s 50` (stylize for more photorealistic)

### DALL-E 3 Specific

- Keep prompts more conversational
- Specify "photorealistic photograph" explicitly
- Include mood and lighting descriptions

### Stable Diffusion Specific

Add to positive prompt:

- `masterpiece, best quality, highly detailed`
- Use specific checkpoint models for photorealism (e.g., Realistic Vision)

### Leonardo AI Specific

- Select "PhotoReal" model for realistic images
- Use "Prompt Magic" for enhanced results
- Select appropriate preset (Cinematic, Photography, etc.)

## Quality Control Checklist

Before using generated images, verify:

- [ ] No anatomical errors (extra/missing limbs, deformed features)
- [ ] Text is not generated in image (or is acceptable)
- [ ] Colors align with brand palette
- [ ] Resolution meets minimum requirements
- [ ] No watermarks or artifacts
- [ ] Mood matches intended use
- [ ] Lighting is consistent and professional
- [ ] Faces look natural (if human subjects)
- [ ] Animals look healthy and happy
- [ ] Background is appropriate for context
